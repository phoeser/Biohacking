Die KI-Funktion läuft direkt in deinem Browser und nutzt deinen eigenen, kostenlosen Google-AI-Studio-Key (Gemini). Der Key bleibt ausschließlich lokal (localStorage) auf deinem Gerät und wird nicht an Dritte weitergegeben.

So bekommst du einen kostenlosen Key: aistudio.google.com/apikey → „Create API key" → kopieren → hier einfügen.
